1.使用左移运算符 << 迅速得出 2 的次方 ​

javascript
1 << 2 // 4, 即 2的2次方
1 << 10 // 1024, 即 2的10次方

// 但是要注意使用场景
a = 2e9 // 2000000000
a << 1 // -294967296

5. 使用~、>>、<<、>>>、|来取整 ​

相当于使用了 Math.floor()

javascript

console.log(11.71 >> 0) // 11
console.log(11.71 << 0) // 11
console.log(11.71 | 0) // 11
console.log(11.71 >>> 0) // 11  注意 >>> 不可对负数取整

6. 使用^来完成值交换 ​

在 ES6 的解构赋值出来之前，用这种方式会更快(但必须是整数)

javascript
// --- before ---
let temp = a
a = b
b = temp // 传统，但需要借助临时变量
b = [a, (a = b)][0] // 借助数组

// --- after ---
let a = 7
let b = 1
a ^= b
b ^= a
a ^= b
console.log(a) // 1
console.log(b)[(a, b)] = // 7
  [b, a] // ES6，解构赋值
