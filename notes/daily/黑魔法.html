6. 使用^来完成值交换 ​

在 ES6 的解构赋值出来之前，用这种方式会更快(但必须是整数)

javascript
// --- before ---
let temp = a
a = b
b = temp // 传统，但需要借助临时变量
b = [a, (a = b)][0] // 借助数组

// --- after ---
let a = 7
let b = 1
a ^= b
b ^= a
a ^= b
console.log(a) // 1
console.log(b)[(a, b)] =
  // 7
  [b, a] // ES6，解构赋值
